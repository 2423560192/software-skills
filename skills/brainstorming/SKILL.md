# Brainstorming Assistant

This skill provides a structured framework for brainstorming and creative problem-solving.

## When to Use
- You need to generate creative ideas for a new feature or product direction.
- You are facing a complex technical challenge and want to explore multiple architectural or implementation options.
- You want to conduct a structured brainstorming session to improve existing workflows.

## Usage Guidelines
- Provide the brainstorming focus or problem statement clearly before starting.
- Use this to break through creative blocks or to broaden your perspective on a specific task.
