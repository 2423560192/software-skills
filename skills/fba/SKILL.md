# FastAPI Practices Assistant (fba)

This skill provides specialized assistance for projects following FastAPI best practices. 

## When to Use
- You are working on a FastAPI project and need guidance on structure, dependency injection, or best practices.
- You want to implement standard patterns common in the `fastapi-practices` ecosystem.
- You need help debugging FastAPI-related components or services.

## Usage Guidelines
- Always ensure you are within a FastAPI-structured project before invoking.
- Use this to keep your code idiomatic and aligned with modern FastAPI standards.
