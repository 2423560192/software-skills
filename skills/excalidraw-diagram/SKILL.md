# Excalidraw Diagram Assistant

This skill helps you generate Excalidraw-compatible diagrams to visualize complex technical concepts.

## When to Use
- You need to explain the system architecture, component relationships, or logical flows visually.
- You want to generate diagram code (e.g., Mermaid or Excalidraw JSON) for documentation or design discussions.
- You need a visual aid to better understand a complex piece of code or process.

## Usage Guidelines
- Provide a clear description of the process or architecture you wish to visualize.
- Use this to create diagrams that can be easily rendered or edited in Excalidraw tools.
